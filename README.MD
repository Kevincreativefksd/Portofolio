SEO Architect Portfolio (Neobrutalism UI)

Portfolio website sederhana berbasis HTML, CSS, dan JavaScript dengan gaya desain Neobrutalism.
Dirancang untuk menampilkan pengalaman sebagai SEO Architect serta studi kasus seperti Zona Ekonomi.

---

 Features

-  Landing page SEO-friendly (meta tag + struktur jelas)
-  Hero section dengan positioning "4+ Tahun Experience"
-  Dark Mode (Neon Green Accent)
-  UI Neobrutalism (border tebal + shadow tajam)
-  Animated Bar Chart (visual growth traffic)
-  Portfolio Page (case study & client success)
-  CTA langsung ke WhatsApp (lead generation)
-  External link ke project (Zona Ekonomi)

---

 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (No Framework)
- Canvas API (untuk chart animasi)

---

 Project Structure

/project
│── index.html          # Landing page
│── portfolio.html      # Halaman portfolio
│── style.css           # Styling (Neobrutalism + Dark Mode)
│── script.js           # Logic + animation + components

---

 How to Use

1. Download / clone repository ini
2. Buka di browser atau editor seperti SPCK / Acode
3. Jalankan "index.html"
